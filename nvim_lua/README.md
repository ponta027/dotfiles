# README

## this document

how to apply nvim config.

## proceduer

```
git clone https://github.com/wbthomason/packer.nvim  ~/.config/nvim/pack/packer/opt/packer.nvim

neovim
:PackerCompile
:PackerInstall
```


### install coc

```
cd ~/.config/nvim/site/pack/packer/start/coc.nvim
nvm install lts
yarn install
```
